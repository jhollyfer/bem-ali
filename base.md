## Customer

1. Deve Cadastrar-se na plataforma
2. Deve Alterar os seus dados
3. Deve Visualizar seus dados

## Proprietário

1. Deve Cadastrar-se na plataforma
2. Deve Alterar os seus dados
3. Deve Visualizar seus dados

## Proprietário x Casas

1. Um proprietário deve cadastrar uma ou mais casas
2. Um proprietário deve atualizar os dados de uma casa
3. Um proprietário deve buscar os dados de uma casa
4. Um proprietário deve buscar todas as casas que cadastrou
5. Um proprietário deve conseguir informar se casa está disponível
6. Um proprietário deve informar uma previsão de quando a casa deverá estar disponível novamente

## Customer (Cliente) x Casas

1. Um cliente deve buscar e visualizar os dados de uma casa
2. Um cliente deve buscar e listar todas as casas cadastradas
3. Um cliente deve buscar casas disponíveis em próximo a sua localização
4. Um cliente deve buscar casas por disponibilidade, localização, preços, quantidade de quartos, etc
5. Um cliente deve mostrar interesse em uma casa
